# 模型对比说明

本文档对比项目中可用的三个 AI 模型的特点和使用场景。

---

## 📊 模型对比表

| 特性 | DeepSeek Chat | DeepSeek Reasoner | GPT-4o Mini |
|------|---------------|-------------------|-------------|
| **图标** | 💬 | 🧠 | 🤖 |
| **提供商** | DeepSeek | DeepSeek | OpenAI |
| **主要特点** | 快速通用对话 | 深度推理思考 | 多模态识别 |
| **响应速度** | ⚡⚡⚡ 极快 | ⚡⚡ 较慢（需要思考） | ⚡⚡ 中等 |
| **成本** | 💰 低 | 💰💰 中等 | 💰💰 中等 |
| **思维链显示** | ❌ 否 | ✅ 是 | ❌ 否 |
| **图片识别** | ❌ 不支持 | ❌ 不支持 | ✅ **支持** |
| **文件上传** | ❌ 不支持 | ❌ 不支持 | ✅ 支持 |
| **支持的文件** | - | - | 图片、PDF、TXT 等 |

---

## 🎯 模型详解

### 1️⃣ DeepSeek Chat 💬

**定位**：快速通用对话模型

**适用场景**：
- ✅ 日常对话聊天
- ✅ 文本问答
- ✅ 代码解释
- ✅ 内容创作
- ✅ 翻译润色

**特点**：
- 响应速度极快
- 成本低廉
- 适合大部分日常使用场景

**不支持**：
- ❌ 图片识别
- ❌ 文件上传
- ❌ 思维链显示

---

### 2️⃣ DeepSeek Reasoner 🧠

**定位**：深度推理思考模型

**适用场景**：
- ✅ 复杂逻辑推理
- ✅ 数学题解答
- ✅ 编程难题
- ✅ 策略分析
- ✅ 需要查看思考过程的场景

**特点**：
- 显示完整的思维链（Reasoning）
- 可以看到模型的思考过程
- 回答更加深入和详细
- 适合需要"慢思考"的问题

**不支持**：
- ❌ 图片识别
- ❌ 文件上传

**界面展示**：
```
┌─────────────────────────────────┐
│ DeepThink                       │
│ 首先，我需要理解这个问题...     │
│ 然后，我应该考虑...             │
│ 最后，得出结论...               │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ 最终回答内容                     │
└─────────────────────────────────┘
```

---

### 3️⃣ GPT-4o Mini 🤖

**定位**：OpenAI 多模态模型（图片识别能力强）

**适用场景**：
- ✅ **图片识别** ⭐ 核心功能
- ✅ 图片内容描述
- ✅ 图表数据提取
- ✅ 截图问题解答
- ✅ PDF 文档分析
- ✅ 代码截图识别

**特点**：
- **真正的图片识别能力**（不仅限于 OCR）
- 可以理解图片的内容、场景、对象
- 支持图片 + 文字混合输入
- 支持 PDF、TXT 等文件上传

**支持的文件类型**：
```
📷 图片: JPG, PNG, GIF, WebP
📄 文档: PDF (提取文本)
📝 文本: TXT, MD, JSON, CSV
💻 代码: JS, JSX, TS, TSX, PY, JAVA, CPP, etc.
```

**图片识别示例**：
- 上传一张风景照 → 模型能描述场景、颜色、物体
- 上传一张图表 → 模型能读取数据和趋势
- 上传一张代码截图 → 模型能解释代码逻辑
- 上传一张表情包 → 模型能理解幽默含义

## 🔄 模型切换

### 如何切换模型？

在输入框上方的下拉菜单中选择：

```
┌─────────────────────────────┐
│ 💬 DeepSeek Chat           │ ← 点击切换
│ 🧠 DeepSeek Reasoner        │
│ 🤖 GPT-4o Mini             │
└─────────────────────────────┘
```

### 自动切换逻辑

- 当选择 **DeepSeek Chat** 或 **DeepSeek Reasoner** 时：
  - ❌ 附件按钮消失
  - ❌ 无法上传文件

- 当选择 **GPT-4o Mini** 时：
  - ✅ 显示附件按钮
  - ✅ 可以上传图片和文档

## 💡 使用建议

### 日常对话 → DeepSeek Chat 💬
```
用户：帮我写一首关于春天的诗
模型：✅ 快速生成优美的诗歌
```

### 复杂推理 → DeepSeek Reasoner 🧠
```
用户：有 100 个囚犯的帽子问题怎么解？
模型：
  [DeepThink 显示]
  首先分析问题...
  然后推导策略...
  
  [最终回答]
  详细的解题步骤...
```

### 图片识别 → GPT-4o Mini 🤖
```
用户：[上传一张图片]
模型：✅ 这是一张办公室场景照片，包含...
      我看到桌上有笔记本电脑、咖啡杯...
```

### 文档分析 → GPT-4o Mini 🤖
```
用户：[上传 PDF] 总结这份报告的要点
模型：✅ [提取 PDF 文本]
      这份报告主要讨论...
```

## ⚠️ 重要说明

### GPT-4o Mini vs GPT-4o

**为什么使用 GPT-4o Mini 而不是 GPT-4o？**

| 特性 | GPT-4o | GPT-4o Mini |
|------|--------|-------------|
| 图片识别 | OCR（仅文字识别） | ✅ **真正的视觉理解** |
| 成本 | 较高 | 较低 |
| 速度 | 较慢 | 较快 |
| 场景理解 | 有限 | ✅ 强大 |

**结论**：GPT-4o Mini 在图片识别方面**更强**，且成本更低。

---

## 📈 成本对比（估算）

| 模型 | 每 1M tokens 成本 | 适合场景 |
|------|------------------|----------|
| DeepSeek Chat | ~￥2 | 高频日常使用 |
| DeepSeek Reasoner | ~￥5 | 偶尔深度思考 |
| GPT-4o Mini | ~￥4 | 需要图片识别时 |

**省钱建议**：
- ✅ 优先使用 DeepSeek Chat 进行日常对话
- ✅ 只在需要看思维过程时用 DeepSeek Reasoner
- ✅ 只在需要处理图片/文件时用 GPT-4o Mini
